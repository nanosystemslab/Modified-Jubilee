# Modified Jubilee  
### Master’s Thesis Design Files & Documentation  

---

## Overview

This repository contains all design files, mechanical models, firmware, scripts, and supporting documentation used in the development of my Master’s thesis:

**“DESIGN, MODIFICATION, AND EVALUATION OF THE JUBILEE  
OPEN-SOURCE MOTION PLATFORM FOR AUTOMATED DIP COATING”**

This repository exists to support replication, modification, and extension of the system in alignment with open-source hardware principles.
